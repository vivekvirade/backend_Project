GET YOUTUBE SUBSCRIBERS

These are the routes in this API

You can try entering each route using the routes given below

  • GET /subscribers in URL : Provides an array of all subscribers in the database.
  • GET /subscribers/names in URL : Provides an array of subscribers with only two fields name and subscribedChannel
  • GET /subscribers/:id in URL: Provides the details of a subscriber with the given ID.
